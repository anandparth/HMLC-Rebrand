# HMLC Rebrand — Case Study

A product design case study documenting the full digital redesign of **Harsh Mann Luxury Consultancy** (hmluxuryconsultancy.com) — a boutique branding agency working with India's top luxury brands.

---

## The Brief

HMLC's brand work had reached a world-class standard. Their website hadn't. The old site leaned on purple gradients, gold clipart, and stock-photo heroes — the visual language of *aspirational* luxury, not actual luxury. The portfolio was buried three clicks deep. The client roster (BMW, ITC, Kotak, 30+ others) was a footnote.

The brief: fix the order of revelation. Let the work do the selling.

---

## What Changed

| Before | After |
|--------|-------|
| Purple + gold template palette | Black, white, and stone |
| 8-item cramped navigation | 5 clear items |
| Portfolio buried 3 clicks deep | Above the fold on homepage |
| Client logos in the footer | Scroll 2, homepage |
| Self-promotional copy | Work-first layout |

---

## The Work

Over **7 months**, the redesign covered:

- **Discovery** — prospect interviews, session recordings, heuristic audit
- **IA restructure** — portfolio-first hierarchy, simplified navigation
- **Visual system** — typography-led, restraint as a design principle
- **Screen design** — homepage, portfolio, services, about, blog
- **Handoff** — component documentation, responsive specs

---

## Case Study Format

The case study is a **single self-contained HTML file** — no build tools, no framework, no dependencies beyond Google Fonts. Open `index.html` in any browser.

Key techniques used:
- Excalidraw-style hand-drawn SVG illustrations (`feTurbulence` + `feDisplacementMap` filters)
- Scroll-reveal via `IntersectionObserver`
- SVG path draw-on animation
- Stacked card testimonials with CSS `:has()` selector
- Animated metric counters
- Marquee keyword band
- Bento grid layout
